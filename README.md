# Customer Shield Analytics — Churn Prediction System

## 📌 Visão do Projeto
Projeto de análise de churn de clientes com foco em prever cancelamentos e melhorar retenção usando dados, SQL e Machine Learning.

## 🎯 Problema de Negócio
Empresas SaaS perdem clientes de forma silenciosa, impactando receita. O objetivo é prever churn antes que aconteça.

## 🧠 Objetivo
- Prever clientes com risco de cancelamento  
- Identificar padrões de comportamento  
- Apoiar estratégias de retenção

## ⚙️ Tecnologias Utilizadas
- Python (Pandas, NumPy)
- SQL
- Machine Learning (Scikit-learn)
- Power BI

## 📊 Entregas
- Limpeza e tratamento de dados  
- Análise exploratória (EDA)  
- Modelo de previsão de churn  
- Insights de negócio

## 💡 Valor para o negócio
Redução de churn e aumento de retenção de clientes com decisões baseadas em dados.
